
# Lifestyle Weight Tracker Dataset Explorer

An interactive data exploration tool for analyzing lifestyle factors and their impact on weight tracking. This project uses **Pandas** for data handling and **D-Tale** to provide a browser-based interface for data visualization.

## Setup and Installation

### Clone the Repository
```bash
git clone git@github.com:mwaura-mwangi/Lifestyle_Weight_Tracker_Dataset.git
cd Lifestyle_Weight_Tracker_Dataset
```

### Create and Activate Virtual Environment
```bash
python -m venv .lifestyle
source .lifestyle/bin/activate  # On Windows: .lifestyle\Scripts\activate
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

## Running the Tracker
To launch the interactive dashboard:
```bash
python lifestyle_tracker.py
```
Check your browser at `http://localhost:40000`.
